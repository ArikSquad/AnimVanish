Plugin that provides large library of pre-made effects to your vanish

### Current effects
- Herobrine effect (Lightning strike and night)
- Particle effect
- TNT effect
- NPC effect (requires [Citizens](https://www.spigotmc.org/resources/citizens.13811/))
- Zombie effect
- Blindness effect
- Sound effect
